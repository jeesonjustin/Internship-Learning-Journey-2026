# Task 13 – Library Book Issue System

## Overview

This project was developed as part of my Full Stack Development Internship during **Week 03**.

The objective of **Task 13** was to develop a Library Book Issue System using **PHP** and **MySQL**, while understanding relational databases and implementing **SQL JOIN** operations to retrieve data from multiple related tables.

---

## Features

- Add New Books
- Add New Members
- Issue Books to Members
- Display Issued Books
- SQL INNER JOIN Implementation
- Foreign Key Relationships
- Dynamic Data Retrieval

---

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- XAMPP
- phpMyAdmin

---

## Project Structure

- `add_book.php` – Add New Books
- `insert_book.php` – Insert Book Records
- `add_member.php` – Add New Members
- `insert_member.php` – Insert Member Records
- `issue_book.php` – Book Issue Form
- `insert_issue.php` – Store Issue Details
- `display.php` – Display Issued Books using SQL JOIN
- `db.php` – Database Connection
- `style.css` – Styling
- `library_db.sql` – Database Export
