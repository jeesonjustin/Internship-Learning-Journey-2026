# Task 11 & 12 – Student Management System

## Overview

This project was developed as part of my Full Stack Development Internship during **Week 03**.

The project combines the objectives of **Task 11** and **Task 12**, where I built a complete Student Management System using PHP and MySQL.

---

## Features

- PHP & MySQL Database Connection
- Student Registration
- Display Student Records
- Update Student Details
- Delete Student Records
- Complete CRUD Operations

---

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- XAMPP
- phpMyAdmin

---

## Project Structure

- `index.php` – Student Registration Form
- `insert.php` – Insert Student Data
- `display.php` – Display Student Records
- `edit.php` – Edit Student Details
- `update.php` – Update Student Information
- `delete.php` – Delete Student Records
- `db.php` – Database Connection
- `style.css` – Styling
- `internship_db.sql` – Database Export
