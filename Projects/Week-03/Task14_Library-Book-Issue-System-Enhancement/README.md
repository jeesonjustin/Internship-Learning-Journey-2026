# Task 14 – Library Book Issue System Enhancement

## Overview

This project was developed as part of my Full Stack Development Internship during **Week 03**.

The objective of **Task 14** was to enhance the existing Library Book Issue System by implementing advanced CRUD operations, search functionality, dashboard statistics, and improving the overall user experience.

---

## Features

- Search Issued Books
- Edit Issue Details
- Update Issue Records
- Delete Issued Records
- Dashboard Statistics
- Total Books Count
- Total Members Count
- Total Issued Books Count
- Improved User Interface
- Responsive Design

---

## Technologies Used

- PHP
- MySQL
- HTML5
- CSS3
- XAMPP
- phpMyAdmin

---

## Project Structure

- `display.php` – Display, Search & Dashboard Statistics
- `edit_issue.php` – Edit Issue Details
- `update_issue.php` – Update Issue Information
- `delete_issue.php` – Delete Issue Records
- `db.php` – Database Connection
- `style.css` – Styling
- `library_db.sql` – Database Export
