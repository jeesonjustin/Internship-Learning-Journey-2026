# Task 04 – Student CRUD Enhanced

## Overview

Enhanced the Student CRUD System by implementing additional features to improve data management, user experience, and application functionality using HTML, CSS, and JavaScript.

This project introduces Local Storage for persistent data, search functionality, course-based filtering, record counting, and improved user interactions through confirmation dialogs and a responsive interface.

---

## Features

- Student Registration
- Display Student Records
- Edit Student Details
- Update Existing Records
- Delete Student Records
- Local Storage Data Persistence
- Search Student Records
- Course-based Filtering
- Record Counter
- Delete Confirmation
- Update Confirmation
- Responsive User Interface

## Workflow

1. Register new student details.
2. Validate user input before submission.
3. Store student records in Local Storage.
4. Display records dynamically in a table.
5. Search students by name.
6. Filter students based on course.
7. Edit and update existing student details.
8. Delete records with confirmation.
9. Automatically maintain the total record count.
