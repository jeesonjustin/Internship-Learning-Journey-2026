# Task 02 – Student Registration Form

## Overview

Developed a responsive Student Registration Form using HTML, CSS, and JavaScript as part of my Full Stack Development Internship.

The project focuses on designing a user-friendly registration form with client-side validation to ensure accurate data entry before form submission.

---

## Features

- Student Name Input
- Age Validation
- Course Selection
- Email Validation
- Required Field Validation
- Responsive Form Layout
- User-friendly Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Concepts Practiced

- HTML Forms
- Input Elements
- Labels
- Form Validation
- JavaScript Events
- DOM Manipulation
- Regular Expressions (Email Validation)
- Responsive Design
