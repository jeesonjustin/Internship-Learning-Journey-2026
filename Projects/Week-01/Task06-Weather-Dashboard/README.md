# Task 06 – Weather Dashboard

## Overview

Developed a responsive Weather Dashboard using HTML, CSS, and JavaScript by integrating the Open-Meteo API. The application allows users to search for weather information by city, view current weather conditions, access a 5-day forecast, and save recent searches using Local Storage.

---

## Features

- Search Weather by City
- Real-time Weather Information
- 5-Day Weather Forecast
- Search History using Local Storage
- Error Handling for Invalid Searches
- Responsive User Interface
- Dark/Light Mode Toggle

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Open-Meteo API
- Fetch API
- Local Storage API

## Workflow

1. Enter the name of a city.
2. Fetch weather data using the Open-Meteo API.
3. Display current weather conditions.
4. Show a 5-day weather forecast.
5. Save recent searches using Local Storage.
6. Display an error message if the city is invalid or data cannot be retrieved.
