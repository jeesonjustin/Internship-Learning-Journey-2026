# Task 03 – Student CRUD System

## Overview

Enhanced the Student Registration Form by implementing a complete CRUD (Create, Read, Update, Delete) system using HTML, CSS, and JavaScript as part of my Full Stack Development Internship.

This project allows users to register student details, display records in a dynamic table, edit existing information, and delete records through an interactive user interface.

---

## Features

- Add New Student Records
- Display Student Records in a Table
- Edit Existing Student Details
- Update Student Information
- Delete Student Records
- Form Validation
- Dynamic Table Rendering
- User-friendly Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript

---

## Concepts Practiced

- CRUD Operations
- DOM Manipulation
- JavaScript Arrays & Objects
- Event Handling
- Form Validation
- Dynamic HTML Table Generation
- Data Updating
- Record Deletion

## Workflow

1. Enter student details in the registration form.
2. Validate the input fields.
3. Add the student record to the table.
4. Edit student information when required.
5. Update the modified details.
6. Delete records with a single click.

---

## Learning Outcome

This project helped me understand the complete CRUD workflow using JavaScript. I learned how to dynamically manipulate the DOM, manage application data, and build interactive web applications without using a backend database.
