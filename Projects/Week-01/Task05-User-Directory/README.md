# Task 05 – User Directory

## Overview

Developed a User Directory web application using HTML, CSS, and JavaScript by integrating a public REST API. The application fetches user information dynamically and displays it in an organized, responsive interface with search functionality and detailed user information.

---

## Features

- Fetch User Data from a REST API
- Display User Information in Dynamic Cards
- Search Users by Name
- View Detailed User Information
- Error Handling for API Requests
- Dark/Light Mode Toggle
- Responsive User Interface

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Fetch API
- REST API

---

## Concepts Practiced

- REST API Integration
- Fetch API
- Asynchronous JavaScript
- Promises
- JSON Data Handling
- DOM Manipulation
- Event Handling
- Search Functionality
- Error Handling
- Responsive Web Design

## Workflow

1. Load the application.
2. Fetch user data from a public REST API.
3. Display user information dynamically as cards.
4. Search users using the search bar.
5. View detailed information for selected users.
6. Handle network or API errors gracefully.

---

## Learning Outcome

This project helped me understand how modern web applications communicate with external APIs to retrieve and display real-time data. I gained practical experience working with asynchronous JavaScript, Fetch API, JSON data, and dynamic DOM manipulation while improving the overall user experience through responsive design and search functionality.
