# Week 03

## Overview

Completed the third week of my Full Stack Development Internship by developing database-driven web applications using **PHP** and **MySQL**. This week focused on establishing database connectivity, implementing complete CRUD operations, working with SQL JOIN queries, enhancing application functionality, and gaining an introduction to the Laravel framework.
